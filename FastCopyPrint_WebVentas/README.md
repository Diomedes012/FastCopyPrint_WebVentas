Descripción del Proyecto
FastCopyPrint es una solución integral diseñada para la gestión de ventas y catálogos de productos. Este sistema web, construido con tecnologías modernas como Blazor busca optimizar el flujo de trabajo en entornos de impresión y copiado, permitiendo una administración eficiente del inventario y una experiencia de usuario fluida.
Mini Manual de Funcionalidades 

Este sistema permite realizar las siguientes operaciones principales:
•	Catálogo Digital: Visualización interactiva de productos y servicios disponibles, facilitando la búsqueda y selección por parte de los usuarios o vendedores.
•	Gestión de Ventas: Registro y procesamiento ágil de órdenes de venta.
•	Administración de Inventario: (Opcional, según tu avance) Control de stock para asegurar la disponibilidad de insumos y productos.
•	Interfaz Intuitiva: La navegación es responsiva y amigable, adaptándose a diferentes dispositivos.
Tecnologías Utilizadas 
•	.NET / C#
•	Blazor (Framework web)
•	MudBlazor (Librería de componentes UI)
•	Base de Datos: SQL Server
